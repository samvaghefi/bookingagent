# BookingAgent

Multi-tenant AI receptionist platform for service businesses.

## Features
- AI-powered voice receptionist
- Automatic appointment booking
- SMS & Email confirmations
- Google Calendar integration
- Multi-business support

## Tech Stack
- Node.js / Express
- PostgreSQL (Supabase)
- Vapi AI
- Twilio
- SendGrid
